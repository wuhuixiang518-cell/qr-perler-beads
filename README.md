# 二维码转拼豆图案转换器

将二维码图片转换为拼豆（Perler Bead / 烫豆）图案的网页工具。纯前端，无需服务器，浏览器打开即可使用。

## 功能

- **上传二维码图片** — 支持任意格式的二维码图片上传
- **自动检测二维码** — 自动定位二维码在图片中的位置、识别版本号（模块数）
- **彩色二维码支持** — 针对绿色等非黑色二维码做了颜色感知灰度化处理，微信截图也能准确识别
- **区域精修** — 通过边缘密度评分精确定位二维码边界，应对复杂背景干扰
- **实时预览** — 调整参数后即时看到拼豆图案变化
- **参数调节**
  - 网格大小（15-80px）
  - 豆子间距（0-20%）
  - 亮度微调（-60 ~ +60）
  - 边框开关
- **PNG 导出** — 一键导出拼豆图案为高清 PNG 图片

## 使用方法

1. 用浏览器打开 `index.html`
2. 点击「上传二维码」选择二维码图片
3. 等待自动检测，查看识别结果（暗模块数量、二维码版本）
4. 在「参数设置」中调节网格大小、豆子间距等
5. 点击「导出 PNG」保存拼豆图案

## 技术原理

### 二维码检测

1. **颜色识别** — 使用 `R+B-G` 颜色感知公式替代标准亮度灰度化，绿色二维码暗模块灰度值约 6，白色模块约 254，分离度高
2. **OTSU 二值化** — 自动计算最佳阈值将图像转为黑白
3. **位置检测** — 通过颜色掩码 + 行列投影粗略定位二维码区域
4. **区域精修** — 提取 n×n 模块网格，计算相邻模块跳变率（边缘密度），排除 Finder Pattern 区域后在 ±10px 范围内搜索最优位置
5. **安全检查** — 精修前后分别提取网格，只有当精修结果暗模块比例更接近 50%（偏差超过 5pp）时才采用，防止误修正

### 拼豆渲染

- 每个 QR 模块映射为一个豆子，豆子内部含 2×2 四宫格（同色，零间隔）
- Finder Pattern 用三种颜色区分（深色/浅色/粉色）
- 支持实时参数调整和 PNG 导出

## 测试效果

- **方形裁剪图** — 713 暗模块（保持原始检测结果）
- **手机微信截图（复杂背景）** — 716 暗模块（精修后），与方形图差异仅 3 模块（0.2%）
