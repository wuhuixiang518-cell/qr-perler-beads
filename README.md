# 二维码转豆豆

将二维码图片转换为拼豆图案的网页工具。纯前端单文件，无需服务器，浏览器打开即用。

## 功能

- **拖拽或点击上传** — 支持任意格式二维码图片，上传后自动转换
- **自动识别二维码** — 自动定位二维码区域、识别版本号（模块数），无需手动裁剪
- **彩色二维码支持** — 针对微信绿色二维码做了颜色感知灰度化，截图也能准确识别
- **区域精修** — 通过边缘密度评分精确定位边界，应对手机截图复杂背景干扰
- **三种配色** — 经典绿（微信风格）、中国红、经典黑白，切换即时生效
- **豆子间距调节** — 0\~15px 滑块，调整豆子之间的间距
- **六种豆子图例** — 转换后展示所用豆子类型及数量，方便备豆
- **PNG 导出** — 一键下载拼豆图案为高清 PNG

## 使用方法

1. 用浏览器打开 `index.html`
2. 拖拽或点击上传区域选择二维码图片
3. 自动检测并转换，右侧显示拼豆图案，左下显示检测信息和豆子图例
4. 可切换配色方案、调整豆子间距，图案即时更新
5. 点击「下载」保存 PNG 图片

## 技术原理

### 二维码检测

1. **颜色识别** — 使用 `R+B-G` 颜色感知公式替代标准亮度灰度化，绿色暗模块灰度值约 6，白色模块约 254，分离度高
2. **Otsu 二值化** — 自动计算最佳阈值将图像转为黑白
3. **位置检测** — 通过颜色掩码 + 行列投影粗略定位二维码区域
4. **区域精修** — 提取 n×n 模块网格，计算相邻模块跳变率（边缘密度），排除 Finder Pattern 区域后在 ±10px 范围内搜索最优位置
5. **安全检查** — 精修前后分别提取网格，只有当精修结果暗模块比例更接近 50%（偏差超过 5 个百分点）时才采用，防止误修正

### 拼豆渲染

- 每个 QR 模块映射为一个豆子，豆子内部为 2×2 四宫格（4 个点各黑或白）
- 16 种组合经旋转等价后归纳为 6 种豆子类型（全白、单黑、相邻双黑、对角双黑、三黑、全黑）
- Finder Pattern 用三种颜色区分（深色 / 浅色 / 粉色），方便定位
- 配色方案和间距修改即时重绘，无需重新转换
